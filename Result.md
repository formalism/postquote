# 実施結果報告書

Spec.mdに基づき、株価取得およびDiscord通知プログラムを実装しました。

## 実装内容

1.  **環境構築**
    - [Bun](https://bun.sh/) を使用した実行環境の構築。
    - `cheerio` ライブラリを使用したHTMLスクレイピングの実装。
2.  **パーサーの実装 (`parser.ts`)**
    - Yahoo!ファイナンスのHTMLから「会社名」「株価」「前日比（金額）」「前日比（率）」を抽出するロジックを実装しました。
3.  **メインプログラムの実装 (`index.ts`)**
    - `settings.conf` から株式コード、**株式数量(AMOUNTS)**、およびDiscord Webhook URLを読み込みます。
    - 各コードに対して非同期でデータを取得・解析します。
    - **取得したデータと数量に基づき、評価額合計と前日比合計を算出します。**
    - Discordの2000文字制限を考慮し、適切にメッセージを分割して送信します。

## 作成ファイル一覧

- `parser.ts`: スクレイピング・解析用モジュール
- `index.ts`: メイン実行プログラム
- `settings.conf`: 設定ファイル（株式コード、Webhook URL）
- `verify.ts`: モックデータによるパース検証スクリプト
- `confirm_4005.ts`: 実データ（保存済みHTML）による検証スクリプト

## 実行方法

```bash
bun run index.ts
```

## 追加機能 (2026-01-31)

`Spec.md`の更新に基づき、以下の機能を実装しました。

### 1. 株式数量（AMOUNTS）の対応
- `settings.conf` に `AMOUNTS` を追加し、各銘柄の保有数量を設定可能にしました。
- 設定読み込み時に、`CODES` と `AMOUNTS` の要素数が一致するかバリデーションを行います。

### 2. 合計評価額・前日比の算出
- 取得した株価と設定された数量から、ポートフォリオ全体の評価額と前日比合計を計算します。
- 全銘柄のデータ取得に成功した場合、Discord通知の末尾に合計情報を追加して送信します。
